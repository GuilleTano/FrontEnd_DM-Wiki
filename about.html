Thank you for visiting this website. Since you're here, I'll tell you a little bit about the site:

This project was born with the purpose of serving as a practice for web development.
It started as just a FrontEnd project that consumed the DAPI API, but for various reasons, it ended up scaling into something bigger.

Currently, the site has its own BackEnd, consisting of a server developed in nodeJS, which relies on a MongoDB database and an S3 bucket in AWS.
